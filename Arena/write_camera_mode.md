# Write_Camera_Mode

Type: `POST`

Allows setting various camera variables.

## Format

`http://127.0.0.1:6721/Write_Camera_Mode/mode={}&&num={}`

### Options

#### `mode={}`

- `pov`
- `level`
- `follow`
- `disc`
- `side`
- `free`
- `api` (no idea what this is)

#### `num={}`

- `0-9` (assumed player data)
